# Penjelasan Algoritma Greedy

Bot menggunakan pendekatan greedy berdasarkan:
- energy management,
- adaptive firepower,
- survival movement,
- aggressive positioning.

Setiap bot memiliki heuristic berbeda:

## TolakAnginBot
Adaptive pressure bot dengan movement berdasarkan energy.

## KacauuwBot
Aggressive chaos bot dengan adaptive dodge.

## VelocityBot
Evasive movement bot dengan fokus survival.

## LarasBot
Balanced adaptive combat bot.

## PlengerBot
Lightweight sniper-survival bot.

---

# Requirement

- .NET SDK 10
- Java 17+
- Robocode Tank Royale GUI 0.30.0

---

# Cara Menjalankan Program

## Menjalankan GUI

```powershell
java -jar robocode-tankroyale-gui-0.30.0.jar
```

---

## Menjalankan Bot

Masuk ke folder bot:

```powershell
cd src/TolakAnginBot
```

Lalu jalankan:

```powershell
./TolakAnginBot.cmd
```

Lakukan untuk bot lainnya.
